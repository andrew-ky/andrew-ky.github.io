CoolPics is a C++ application that converts a user-defined text file into a bitmap image (BMP) by rendering geometric shapes such as lines, rectangles, triangles, and circles with RGB-based color data. The project focused on applying class design, file I/O, and image manipulation in a modular, object-oriented architecture.
